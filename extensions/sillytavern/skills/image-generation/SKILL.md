# Image Generation

You can help users generate images for their SillyTavern experience.

## Capabilities

- **Character Portraits**: Generate character appearance images
- **Scene Illustrations**: Create scene and environment art
- **Expression Sheets**: Generate character expressions
- **Style Matching**: Maintain consistent art style
- **Prompt Crafting**: Create effective image prompts

## Prompt Structure

### Basic Format
```
[subject], [details], [style], [quality tags]
```

### Example
```
1girl, long silver hair, blue eyes, elegant dress, 
fantasy setting, detailed background,
masterpiece, best quality, highly detailed
```

## Character Portrait Tips

### Physical Features
- Hair: color, length, style
- Eyes: color, shape, expression
- Face: shape, features, makeup
- Body: build, height, posture
- Clothing: style, colors, accessories

### Style Tags
- Anime/manga style
- Realistic/photorealistic
- Painterly/artistic
- Pixel art
- Chibi/cute

### Quality Tags
- masterpiece
- best quality
- highly detailed
- sharp focus
- professional

## Scene Generation

### Environment Elements
- Location type (indoor/outdoor)
- Time of day
- Weather/atmosphere
- Lighting conditions
- Key objects/features

### Composition
- Camera angle
- Focus point
- Depth of field
- Framing

## Negative Prompts

Common exclusions:
- low quality, worst quality
- blurry, out of focus
- bad anatomy, bad hands
- extra limbs, missing limbs
- watermark, signature

## Integration with SillyTavern

1. Use character descriptions for consistent portraits
2. Match scene descriptions to story moments
3. Generate expression variations
4. Create location reference images
5. Maintain style consistency across images

# Group Chat Management

You can help users manage group chats with multiple characters in SillyTavern.

## Capabilities

- **Group Setup**: Configure multi-character conversations
- **Turn Management**: Control speaking order and frequency
- **Character Dynamics**: Define inter-character relationships
- **Conflict Resolution**: Handle character disagreements
- **Scene Coordination**: Manage group scenes effectively

## Group Chat Configuration

### Character Selection
- Choose complementary personalities
- Balance active vs. passive characters
- Consider relationship dynamics
- Limit group size for coherence (3-5 recommended)

### Turn Order Options
- **Round Robin**: Each character speaks in order
- **Random**: Random character selection
- **Weighted**: Based on relevance/activity
- **Manual**: User selects next speaker

### Activation Settings
- Response probability per character
- Trigger keywords for specific characters
- Silence conditions
- Interruption rules

## Character Dynamics

### Relationship Types
- Allies/friends
- Rivals/enemies
- Romantic interests
- Mentor/student
- Strangers

### Interaction Patterns
- Who initiates conversations
- Who responds to whom
- Conflict triggers
- Alliance formations

## Best Practices

### Scene Management
1. Establish clear scene context
2. Give each character distinct moments
3. Use character tags for clarity
4. Manage pacing carefully
5. Allow natural conversation flow

### Avoiding Issues
1. Prevent character voice blending
2. Maintain distinct personalities
3. Handle absent characters gracefully
4. Manage context window carefully
5. Use summaries for long group chats

## Group Chat Prompts

### System Prompt Additions
- Define group dynamics
- Specify interaction rules
- Set scene context
- Establish speaking conventions

### Character-Specific
- How they address others
- Their role in the group
- Typical reactions
- Relationship notes
